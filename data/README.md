# Data Directory

## Files

### schemes.csv
Contains government scheme information with columns:
- `id`: Unique scheme identifier
- `name`: Scheme name
- `description`: Brief description
- `category`: Scheme category (Agriculture, Healthcare, Education, etc.)
- `min_age`, `max_age`: Age eligibility range
- `max_income`: Maximum income eligibility
- `state`: Applicable state (or "All" for central schemes)
- `deadline`: Application deadline
- `benefits`: Scheme benefits

### users_sample.json
Sample user profiles for testing recommendations.

## Data Sources
- Government of India official portals
- State government websites
- MyScheme portal (myscheme.gov.in)
