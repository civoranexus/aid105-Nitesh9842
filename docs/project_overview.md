# SchemeAssist AI - Project Overview

## Project ID: AID105
## Organization: Civora Nexus Pvt. Ltd.

## Problem Statement
Many citizens are unaware of government schemes they are eligible for, leading to underutilization of welfare programs.

## Solution
SchemeAssist AI is an intelligent recommendation system that:
1. Collects user profile information
2. Matches users with eligible government schemes
3. Provides personalized recommendations
4. Sends alerts for upcoming deadlines

## Key Features
- **AI-Powered Matching**: Uses eligibility scoring algorithm
- **Real-time Updates**: Dynamic scheme database
- **Alert System**: Deadline notifications
- **Report Generation**: Downloadable recommendation reports

## Technology Stack
- **Backend**: Python, Flask
- **Frontend**: HTML, CSS, JavaScript
- **Data**: CSV, JSON
- **AI/ML**: Rule-based + future ML integration

## Target Users
- Rural and urban citizens
- Government welfare officers
- NGOs and social workers
